### <h1 align="center"> AluraBooks </h1>
<p align="center">Este projeto "AluraBooks" foi criado com foco em uma interface intuitiva para promover livros de tecnologia. Usando a técnica mobile first, o design prioriza dispositivos móveis, garantindo uma experiência fluida em telas pequenas antes de ser adaptado para dispositivos maiores, mantendo a responsividade e acessibilidade.</p>

### <h2 align="center">Funcionalidades</h2>
<p align="center"> - Menu de navegação interativo com categorias de livros</p>
<p align="center"> - Carrosséis de livros para lançamentos e mais vendidos</p>
<p align="center"> - Seção de recomendações e autoras em destaque</p>
<p align="center"> - Formulário para cadastro de e-mail</p>
<p align="center"> - Rodapé com links para os serviços do Grupo Alura</p>

### <h2 align="center">Tecnologias Utilizadas</h2>
<p align="center"> - HTML5: Estrutura do conteúdo</p>
<p align="center"> - CSS3: Estilização e responsividade</p>
<p align="center"> - Carrosséis interativos com Swiper.js</p>
